# Biosignals GTM

A comprehensive go-to-market (GTM) strategy and Next.js web application for biosignals technology.

## Overview

This repository contains both the strategic planning resources and the web application for bringing biosignals technology to market. It combines GTM strategy documents with a modern Next.js application for marketing and business development.

## Project Structure

```
biosignals-gtm/
├── README.md
├── package.json         # Next.js application dependencies
├── next.config.mjs      # Next.js configuration
├── tailwind.config.ts   # Tailwind CSS configuration
├── tsconfig.json        # TypeScript configuration
├── app/                 # Next.js app router pages and API routes
├── components/          # React components
├── lib/                 # Utility functions and configurations
├── hooks/               # Custom React hooks
├── styles/              # Global styles and CSS
├── public/              # Static assets and images
├── data/                # Data files and content
├── docs/                # Documentation and strategy documents
├── research/            # Market research and analysis
├── strategy/            # GTM strategy and planning
├── assets/              # Marketing assets and materials
└── implementation/      # Implementation plans and resources
```

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

### Build
```bash
npm run build
```

## GTM Strategy Components

This repository includes dedicated folders for comprehensive go-to-market planning:
- **docs/**: Technical documentation and specifications
- **research/**: Market analysis and competitive research
- **strategy/**: GTM strategy and positioning documents
- **assets/**: Marketing collateral and brand assets
- **implementation/**: Project plans and execution frameworks
