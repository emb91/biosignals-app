# Arcova logo

The Arcova mark is **five equal-width bars tracing a soft triangle** — flat
bottoms, rounded fading tops. It reads as an "A" / an aurora of signal without
being a literal letter. The primary brand expression is the **teal-to-pink**
gradient. On the landing page the mark is used bare (no background); the
**navy squircle** app icon is the favourite for icon contexts.

## Palette

| Token | Hex | Use |
| --- | --- | --- |
| Navy | `#003344` | Icon background, dark surfaces |
| Ink | `#0d3547` | Wordmark on light |
| Teal | `#00a4b4` | Primary brand teal (gradient base) |
| Teal-deep | `#006c79` | Gradient base shadow |
| Mint | `#8cd9c9` | Gradient mid |
| Pale pink | `#f6cdda` | Gradient tip (teal-to-pink only) |
| Lilac | `#d8c6e6` | Bridge between teal and pink (avoids a muddy middle) |

**Gradients**
- *Teal-to-pink* — one wash across all five bars, diagonal `#00a4b4 → #7fd8cd → #d8c6e6 → #f6cdda` (`userSpaceOnUse`, so it spans the whole mark, not each bar).
- *Teal* — per-bar vertical `#006c79 → #00a4b4 → #2bbfb8 → #7adfd2 → pale`.

## Wordmark

"arcova", lowercase, **Quicksand weight 500**, letter-spacing `-0.01em`. Ink on
light, white on dark. In the landing lockup the bare mark sits close to the word.

```bash
node branding/logo/generate-assets.mjs
```

## Using it in the app

```tsx
import { Logo } from "@/components/logo"

<Logo badge="none" size={28} />     // bare curtain + "arcova" — landing
<Logo badge="navy" />                // navy squircle + wordmark — app chrome
<Logo variant="icon" badge="navy" /> // squircle badge alone
```

`badge`: `navy` (default) · `white` · `none`. `mark`: `teal-pink` (default) · `teal` · `white` · `navy`.

The favicon / app icons are wired in `app/layout.tsx` (`metadata.icons`) to `/brand/*`.

## Clear space & minimum size

- Keep clear space around the mark equal to the width of one bar.
- Minimum mark size ~20px; the favicon is generated down to 16px but prefer 24px+ in UI.
- The app icon already has built-in padding — don't wrap the bare mark in a second square.

## Don'ts

- Don't recolour the bars outside the palette, or flip the gradient direction.
- Don't bring back the old concentric-ring / orb treatment.
- Don't stretch, rotate, or drop-shadow the mark.
- Don't set the wordmark heavy — Quicksand 500 is the look.
